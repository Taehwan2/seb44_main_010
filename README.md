# MoneyMade

## Project 
### 개요
- 사용자가 자신의 자산과 입출금을 관리할 수 있는 서비스 ‘💰MoneyMade‘
- 내가 가지고있는 자산(개인계좌, 투자계좌)을 한눈에 보는 서비스 구축
- 내가 사용한 소비내역 입력을 통한 지출계획을 한눈에 보는 서비스 구축


## Member
|이름|역할|담당|이름|역할|담당|
|--|--|--|--|--|--|
|이은호|BE 팀장| AWS, 배포 자동화, 회의록 등 |정지은|FE 팀장| 소비 페이지 |
|장태환|BE 팀원| 기능 구현 |채명수|FE 팀원| - reCAPTCHA를 이용해 로그인 시 로봇인지 아닌지 판별 <br> - 회원가입시 입력하는 아이디인 메일을 통해 인증번호가 가고 그 인증번호와 일치 해야지만 회원가입이 되게 로직 구현 <br> - 전역으로 사용되어져야 했던 프로필 데이터를 서버에서 받은 뒤 Redex에 저장 후 사용 <br> - 메인페이지에서 휠을 한번만 내려도 다음 섹션으로 넘어갈 수 있게 기능 구현 <br> - dialogflow를 이용한 챗봇 UI 구현 <br> - Netlify를 이용해 배포|
|백지희|BE 팀원| 챗봇 담당 |정태현|FE 팀원| 자산 페이지|
